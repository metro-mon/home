Welcome

Welcome to Quag's, a Minecraft Server centered around the Minecraft Transit Railway (MTR) mod.

About the Server

We are the first Mohist 1.16.5 server that uses the MTR Mod. Mohist combines the Forge modloader as well as Bukkit, Spigot and Paper plugins. We are currently on soft opening as we build more and more of the map.

About the Server Map

Our server is a Minecraft recreation of a Pokemon Region map with some changes and new regions. We use the MTR mod to construct and operate train stations. Furthermore, we use more plugins and mods such as Create, Moving Elevators and PowerRanks.
